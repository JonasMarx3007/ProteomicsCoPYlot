# ProteomicsCoPYlot

## Install

### 1. Clone the project
git clone https://github.com/JonasMarx3007/ProteomicsCoPYlot.git
cd ProteomicsCoPYlot

### 2. Setup backend
cd backend
py -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt

### 3. Setup frontend
cd frontend
npm install

### 3. Run
cd ProteomicsCoPYlot
run_app.bat
